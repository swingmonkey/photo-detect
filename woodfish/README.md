# 电子木鱼 - 积功德

一个简单的 2D 网页小游戏，点击木鱼积攒功德。

## 功能特性

- ✨ 点击木鱼或按空格键积功德
- 🔔 木鱼敲击音效（Web Audio API）
- 📊 实时功德计数
- 🔄 自动敲击模式
- 💾 本地数据持久化
- 📱 响应式设计，支持移动端

## 使用方法

1. 点击木鱼区域或按空格键敲击木鱼
2. 每次敲击功德+1
3. 使用控制按钮管理游戏：
   - 重置计数
   - 开关音效
   - 自动敲击

## 技术栈

- 纯 HTML/CSS/JavaScript
- Web Audio API 音效
- LocalStorage 数据持久化
- SVG 矢量图形

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。
